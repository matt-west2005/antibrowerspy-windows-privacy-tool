## Overview

AntiBrowserSpy is a Windows-oriented privacy utility for users who want to reduce browser tracking and control the traces left behind in day-to-day browsing. It brings together anti-fingerprinting options, telemetry blocking, and cleanup tools so multiple privacy tasks can be managed from a single interface.

The app is aimed at people who want tighter control over browser artifacts, profile data, and tracking signals without relying on several separate tools. With runtime monitoring and a configuration wizard included, it becomes easier to inspect settings, apply changes, and keep browser cleanup tasks organized.

## Usage

A typical workflow looks like this:

1. Open AntiBrowserSpy.
2. Review the current privacy and tracking settings.
3. Choose the protections you want to enable, such as anti-fingerprinting or telemetry blocking.
4. Run cleanup options for cookies, browser artifacts, or profile data when needed.
5. Monitor status changes during runtime to confirm the selected configuration is active.

If your build includes browser profile cleanup controls, use them after backing up any data you want to keep. For best results, apply settings before starting a browsing session rather than after.
